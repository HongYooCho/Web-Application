body 태그에는 내용, head 태그에는 이 문서를 설명하는 정보가 들어간다.  

body tag  
- <strong> 강조 태그
- <a> 링크 태그
- <li> 리스트
- <ol> 순서가 있는 리스트(ordered list)
- <ul> 순서가 없는 리스트(unordered list)

head tag  
- <meta charset="utf-8"/> utf라는 문자코드로 저장되어 있는 문서
- <title> 웹페이지 제목 설정
